# 게시판 리스트 구현 요구사항

## 프로젝트 정보
- 프레임워크: React
- 파일명: Board.jsx
- 필요한 라이브러리: 이미 설치됨 (추가 설치 불필요)

## 화면 구성

### 제목
- 게시글 리스트

### 테이블
- Bootstrap 사용
- 배경색: 밝은 계열
- 테이블 헤더: dark 모드
- 정렬: 중앙 정렬
- 테이블 컬럼 순서: 번호 → 제목 → 조회수 → 작성일

### 게시글 쓰기 버튼
- 위치: 테이블 상단 우측
- 색상: 녹색 계열

### 페이지당 개수 선택
- selectbox 사용
- 옵션: 10개, 20개, 30개
- 기본값: 10개
- 선택 시 서버에 재요청하여 리스트 갱신

## 데이터 통신

### API 정보
- 통신 방식: axios
- 서버 주소: http://222.239.249.197:9090/api/v1/board
- 파라미터:
  - size: 한 페이지당 게시글 수
  - page: 페이지 번호 (0부터 시작)

### 서버 응답 데이터 형식

## 라우팅

### Router 설정
- React Router data mode 사용
- lazy 옵션으로 데이터 로드
- loader 함수에서 서버 데이터 fetch
- 페이지 로드 시 바로 화면 렌더링

## 페이징 컴포넌트
- 별도 컴포넌트로 분리
- 위치: components/ 폴더
- Bootstrap 사용
- 블럭당 페이지 번호: 10개씩 표시
- 페이지 번호가 10개 미만일 경우: 가능한 숫자까지만 표시
